# TaskFlow v0.1 Demo 指南

## Demo 目的

体验 TaskFlow 的核心任务管理功能，快速验证产品设计。

## Demo 步骤

### 步骤 1：启动应用

```bash
npm install
npm run dev
```

打开浏览器访问 http://localhost:3000

### 步骤 2：注册账户

1. 点击「注册」按钮
2. 输入任意邮箱和密码（演示模式无需真实邮箱）
3. 点击注册后自动跳转到 Dashboard

### 步骤 3：创建任务

1. 点击「+ 新建任务」按钮
2. 填写任务信息：
   - 标题：完成项目文档
   - 描述：整理所有模块的 API 文档
   - 状态：待办
   - 优先级：高
   - 截止日期：选择一个未来日期
   - 标签：文档
3. 点击「创建」

重复创建 3-5 个不同状态和优先级的任务。

### 步骤 4：任务筛选

- 使用搜索框按关键词筛选
- 使用状态下拉按状态筛选（待办/进行中/已完成）
- 使用优先級下拉按优先级筛选（高/中/低）

### 步骤 5：更新任务状态

- 点击任务卡片的「→」按钮切换状态：
  - 待办 → 进行中 → 已完成 → 待办

### 步骤 6：编辑任务

- 点击任务卡片的「···」菜单
- 选择「编辑」
- 修改任务信息后保存

### 步骤 7：删除任务

- 点击任务卡片的「···」菜单
- 选择「删除」
- 确认删除

### 步骤 8：查看统计面板

Dashboard 顶部显示 6 个统计指标：
- 全部任务数
- 待办数量
- 进行中数量
- 已完成数量
- 逾期数量
- 完成率

### 步骤 9：退出登录

- 点击导航栏右侧「退出」按钮
- 返回首页
